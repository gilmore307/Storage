# Agent Global Memory Snapshot

This snapshot contains the workspace-level memory and identity files for the agent.

## Included
- `AGENTS.md` — workspace operating rules
- `SOUL.md` — identity / tone / execution philosophy
- `USER.md` — stable user preferences
- `TOOLS.md` — environment-specific operational notes
- `HEARTBEAT.md` — heartbeat instructions
- `IDENTITY.md` — lightweight identity metadata
- `memory/` — daily and topic-specific memory files

## Not included
- project-local docs like project `README.md`, `CURRENT_STATE.md`, handoff files, or project-specific runbooks
- missing files that do not currently exist in the workspace (for example `MEMORY.md` is not present right now)

## Purpose
This export is for understanding the agent's global memory structure rather than any one project's internal documentation.
